## Current project

### [Lysis](https://github.com/Blastrum/lysis)

Lysis is a cross-model review loop for AI-generated work.

It lets one model create, another model review, and repeated findings become reusable checks.

It is built for people who already use AI heavily, but still want better quality control for:

- code changes
- plans and specs
- strategy documents
- marketing copy
- other text-based deliverables
